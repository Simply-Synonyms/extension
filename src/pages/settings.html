Simply Synonyms Settings

Settings on this page are saved only to your local browser profile, not to your Simply Synonyms account. You'll have to reload each page after changing a setting for the setting to take effect!

Disable Simply Synonyms

Enabling this option will disable the double-click synonym popup.

Only Enable on Editable Text

Enabling this option will disable the double-click popup on all text that is read-only. It will still be enabled on inputs, documents, text boxes, etc.

Disabled Sites List

Add one website per line like "mail.google.com" or "github.com". Simply Synonyms will be automatically disabled for each website on this list (You can also disable it for an individual domain through the right-click menu on any site).
